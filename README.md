# Projeto Lutar e Vencer Jiu Jitsu

Projeto social criado para ajudar criaças através de aulas de Jiu Jitsu

## Sobre

A API é desenvolvida em Nodejs, utlizando Sequelize para comunicação com o banco Postgres.
O sitema é desenvolvido pra controle de alunos, matricula, frequência e escala dos organizadores.

## Instalação

Totalmente desenvolvido utilizando o [yarn](https://yarnpkg.com/).

Instalacão dos modulos:
```bash
yarn
```

Rodando em modo desenvolvimento:
```bash
yarn dev
```

Gerando build:
```bash
yarn build
```

## Observação

Todas as configurações de conexão com o banco de dados, endereco e porta da aplição devem ser configuradas no .env
